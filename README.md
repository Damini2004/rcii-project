# Researcher Connect Innovation & Impact (RCII) Website

## Overview
A world-class, premium research consulting and academic services website built with React.js, Tailwind CSS, and React Router DOM.

## Technology Stack
- **React 18** — Component-based frontend
- **React Router DOM v6** — Client-side routing
- **Tailwind CSS v3** — Utility-first styling
- **Vite** — Build tool and dev server

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
